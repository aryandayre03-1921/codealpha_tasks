🎬 Exploratory Data Analysis of TMDB Movies Dataset

This project performs Exploratory Data Analysis (EDA) on the TMDB Movies dataset to uncover meaningful insights about the film industry.
Using Python-based data analysis and visualization tools, the project investigates patterns in movie ratings, budgets, revenue, runtime, and language distribution.

The goal is to better understand industry trends, audience preferences, and financial performance of movies through data-driven analysis.

🛠️ Tools & Technologies

The following tools and libraries were used for data analysis and visualization:

• Python – Programming language used for data analysis

• Pandas – Data cleaning and manipulation

• NumPy – Numerical computations

• Matplotlib – Data visualization

• Seaborn – Statistical data visualization

🔎 Exploratory Data Analysis

The analysis focuses on understanding:

• Data structure and variables in the dataset

• Relationships between budget and revenue

• Distribution of movie ratings and runtime

• Language dominance in movie production

• Detection of outliers and blockbuster movies

• Various visualizations such as bar charts, histograms, scatter plots, and count plots were used to explore these insights.

📊 Key Insights from the Dataset
⭐ Movie Ratings

Most movies have ratings between 6 and 8, indicating that the majority of films receive moderately positive audience feedback.

💰 Budget vs Revenue

There is a positive relationship between budget and revenue, suggesting that higher-budget movies tend to generate greater box office returns.

⏱ Movie Runtime

Most movies fall within the 90–120 minute runtime range, which aligns with the typical length of feature films.

🌍 Language Distribution

English is the dominant language in movie production within the dataset, reflecting the global influence of English-language films.

🚀 Revenue Outliers

A small number of movies generate extremely high revenue compared to their budgets, indicating the presence of major blockbuster hits.

📈 Conclusion

This exploratory data analysis revealed several important trends in the movie industry:

• Strong relationships between production budget and revenue generation

• Insights into audience preferences through ratings and popularity

• Identification of common runtime patterns in films

• Understanding of language dominance in movie production

Detection of high-revenue blockbuster outliers

Overall, the analysis demonstrates how data analytics can help uncover meaningful patterns in the entertainment industry, providing valuable insights into film production and audience behavior.
